#+TITLE: Personal NixOS Configuration
#+AUTHOR: lopl
#+DESCRIPTION: A unified NixOS configuration for multiple devices.

This repository manages all my NixOS devices. It's designed to be a single source of truth, ensuring that whether 
I am on my main workstation, laptop or home server, the environment is consistent, reproducible, and easy to update.
** Multiple devices with one flake

This setup uses nix flakes, every dependency is pinned in =flake.lock= and the same repository manages  multiple system configurations as defined inside =flake.nix=. Each machine is recognized by his hostname. Home Manager is used to handle the user level configuration.

** How to Update
Run =nix flake update= to pull the latest versions of NixOS and community modules.

Run =rebuild= (local) or =remote-rebuild= (remote server) to apply changes made to the dotfiles.
